# Library
OOP 4_rd sem java project
Издание
    Свойства:
- id;
- ISBN;
- язык;
- дата публикации;
- издательство;
- название;
- количество страниц;
- количество копий в наличии в библиотеке;
- аннотация;
- возрастное ограничение;
- рейтинг (формируется как среднее арифметическое оценок из отзывов читателей);
- формат (мягкий / твердый переплет);
- библиографическая запись;
- список жанров;
- список отзывов;
- список действий, связанных с этим изданием.

Книга
  Свойства:
- автор.
    Действия над книгой:
- похожие книги;
- просмотр превью.
    
Периодическое издание

Действие над изданием
  Свойства:
- читатель;
- издание.

Бронирование
  Свойства:
- последний день бронирования;
- таймер.

Выдача
  Свойства:
- последний день на руках.
- таймер.

Таймер будет использован для выполнения необходимых действий по истечению времени.

Возврат
  Свойства:
- дата возврата;
- статус (просрочен / нет).

Жанр
  Свойства:
- id;
- название;
- список изданий.

Издательство
  Свойства:
- id;
- название;
- описание;
- список изданий;
- адрес;
- префикс ISBN.

Отзыв
  Свойства:
- id;
- читатель;
- издание;
- дата;
- рейтинг;
- содержание.

Автор
  Свойства:
- id;
- ФИО;
- описание;
- фото;
- список книг.

Пользователь
В веб-приложении будут существовать три типа ролей: читатель, библиотекарь и администратор.

  Свойства пользователя:
- id;
- фамилия;
- имя;
- отчество;
- дата рождения;
- email;
- пароль;
- активирован или нет;
- заблокирован или нет;
- дата окончания блокировки.
  Доступные действия:
- просмотреть профиль;
- выйти из аккаунта.

  Возможности администратора:
- просмотр списка библиотекарей;
- добавить / изменить данные / удалить библиотекаря;
- все действия библиотекаря.

  Возможности библиотекаря:
- просмотр список пользователей, ожидающих подтверждения регистрации, и зарегистрированных пользователей;
- подтверждение регистрацию / изменить данные / удалить / добавить читателя;
- добавить (возможность заполнить известные данные по API Google Books) / изменить данные / удалить издание;
- оформить выдачу / возврат издания, отменить любое действие;
- просмотр списка всех действий за определенный период;
- блокировка аккаунта читателя за нарушение правил (издания возвращены по истечению требуемого срока определенное число раз за фиксированный промежуток времени);
- добавить / изменить / удалить жанр, издательство, автора;
- удаление отзыва.

  Возможности читателя:
- просмотр содержимого следующих разделов веб-приложения: каталог, последние просмотры, забронированные издания, заказанные издания, избранное, издания на руках, выданные издания за определенный период, оставленные отзывы, новые поступления, рекомендации;
- добавить в “избранное”;
- забронировать на определенный период, если издание имеется в наличии, или оставить заявку (придет уведомление на электронную почту, как только в библиотеке появится свободный экземпляр);
- отмена бронирования;
- поиск в каталоге по различных свойствам;
- оставить / удалить отзыв (по изданию, которое брал);
- изменить email (изменение считается подтвержденным после перехода по ссылке из письма) и пароль.

  Возможности неавторизованного пользователя (гостя):
- регистрация путем ввода ФИО, email, даты рождения и пароля (регистрация считается пройденной после ее подтверждения библиотекарем либо администратором, пользователь уведомляется об этом по электронной почте);
- авторизация путем ввода email и пароля;
- просмотр содержимого следующих разделов: каталог, последние просмотры, новые поступления;
- поиск в каталоге по различных свойствам.
